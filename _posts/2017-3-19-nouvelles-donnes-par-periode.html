--- layout: post title: Données hebdomadaires sur l'équité de temps de parole ---

Nous venons de mettre à disposition de nouvelles données sur notre page JSON API : nous disposons maintenant d'un fichier JSON par semaine (environ) récapitulant le temps d'antenne par candidat, ce qui permet de générer très simplement des graphiques régulièrement sur l'évolution du temps de parole semaine après semaine pendant cette campagne présidentielle 2017. Une nouvelle page permet une survol rapide de ces nouvelles données.
